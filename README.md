# RemiDe PDF Generator

Генератор PDF-презентаций из Markdown по дизайн-системе из Figma.

## Как пользоваться

1. Откройте веб-интерфейс
2. Вставьте текст в формате Markdown
3. Нажмите "Сгенерировать PDF"
4. Скачайте готовую презентацию

## Формат Markdown

```markdown
# Заголовок слайда
## Подзаголовок

Текст слайда.

**$120B** — Trade finance — gap annually
**34.2%** — CBR decline — (2011-2022)
```

- `#` — создаёт новый слайд с заголовком
- `##` — подзаголовок
- `**число** — описание — детали` — факт-карточка
- `{accent}текст{/accent}` — акцентный цвет в заголовке

## Переменные окружения

| Переменная | Описание |
|---|---|
| `FIGMA_TOKEN` | Personal Access Token из Figma |
| `FIGMA_FILE_KEY` | Ключ файла из URL Figma |
| `FIGMA_CACHE_TTL` | TTL кеша токенов в секундах. `0` (по умолчанию) — без кеша, изменения из Figma применяются сразу |

## Локальный запуск

```bash
pip install -r requirements.txt
python app.py
```

Откройте http://localhost:8000
